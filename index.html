"Pseudoclases en CSS"

Para mas informacion de Pseudoclases visitar el siguiente link https://developer.mozilla.org/es/docs/Web/CSS/Pseudo-classes

Las pseudoclases brindan estilo a elementos "ocultos"(elementos que no estan en la estructura del doc. html). En el archivo estilo.css hay unas pseudoclases que definen el color de un link sin visitar y visitado. (:link y :visited

---Pseudoclases ":hover, :active, :focus".---

Hover es para brindar estilo cuando pasas el mouse por encima de un elemento

Active es para brindar estilo cuando hasces clic en un elemento

Focus es para brindar estilo cuando a un elememto mientras esta en un foco


Datos personales Nombre Apellidos

---Pseudoclases ":first-child, :last-child, :nth-child".---

First-child es para brindar estilo a el primer elemento de una lista ul

Last-child es para brindar estilo a el ultimo elemento de una lista ul

Nth-child es para brindar estilo a elementos que coincidan con la condicion an+b-1, siendo n un numero natural en una lista u (una pseudoclase complicada)
